# FM ValueScout — Project Concept

FM ValueScout is a desktop companion application for Football Manager players who use moneyball-style scouting — identifying players who are statistically excellent yet undervalued by the transfer market. Built with Tauri v2 and Svelte 5, it replaces the manual spreadsheet workflow that FM moneyball players have historically relied on, ingesting player data via CSV exports from the game and providing purpose-built analysis, ranking, and squad management in a fully offline, locally persisted application.

The tool accumulates season-by-season snapshots into a longitudinal SQLite database, computes percentile scores across 80+ statistical metrics organized into eight categories (Attacking & Finishing, Creativity & Chance Creation, Transition & Ball Progression, Defensive Actions, Aerial Presence, Goalkeeping & Shot Stopping, Discipline & Error Margins, and Match Impact & Availability), and evaluates players against role-specific weighted metric sets called archetypes. Its core features include a scouting module with a role search that surfaces the top three signings ranked by either pure quality or a value-adjusted score that rewards undervalued talent, a squad module with a visual tactics board and lineup optimizer, and a player profile page offering a career timeline and full metric breakdowns. Default archetypes cover all standard positions with in-possession and out-of-possession variants, while customizable weights and filters let power users adapt the system to their own tactical philosophy.
